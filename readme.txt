ТЗ: Создать web-приложение "Фильмотека"
Для реализации проекта:

1) создать БД films.db с применением ORM (sqlalchemy), в которой 2 сущности: каталог (catalog) и
фильмы (films) связаны между собой. 
Создать еще 1 сущность - пользователей (users)

2) предусмотреть регистрацию и авторизацию пользователей

Для пользователей фильмотеки предусмотреть следующие возможности (п.3, 4, 5):

3) незарегистрированные пользователи должны видеть список всех фильмов, а так же этот список в 
отсортированном виде (по названию фильма/сериала и по рейтингу)

4) Зарегистрированные пользователи могут читать более точное описание фильмов.

5) Администратор БД может осуществлять следующие функции:

Работа с фильмами: 
  - просматривать таблицы
  - добавлять/ редактировать/ удалять фильмы/ сериалы
  - загружать файлы изображений фильмов/ сериалов

	Запуск - main.py, переход на http://127.0.0.1:5000/

	Алгоритм дейсвий после запуска программы:
1.Вы - незарегистрированный пользователь. Ваше меню:
   - Главная (список всех фильмов/сериалов)
   - Отсортировать список фильмов/сериалов по названию
   - Отсортировать список фильмов/сериалов по рейтингу

2. Пройдите регистрацию, например:
   email: aaaa@mail.ru
   пароль: 1234
   имя: Второй пользователь
   немного о себе: люблю боевики (необязательное поле)

3. Авторизуйтесь с этим e-mail и паролем

4. Вы - авторизованный пользователь - вы можете читать более подробное описание фильмов и сериалов. 

5. Выйдите, щелкнув по Второй пользователь Выход в правом верхнем углу

6. Авторизуйтесь как Администратор БД (id=1 в таблице users):
   email: admin@mail.ru
   пароль: admin

7. Возможности Администратора БД:
   Добавление/ редактирование/ удаление фильма/ сериала (таблица films):

      - Просмотр таблицы films (фильмы) с возможностью редактирования и удаления записей
      - Загрузка файлов изображений для таблицы films (фильмы) (размер изображений при сохранении будет изменен на 200 х 250)
